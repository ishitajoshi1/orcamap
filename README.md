# OrcaSoundViz WebApp
Node.JS site for prototyping map app using mapbox. 

## Setup

1. Clone the repo.
2. Install Nodejs:Express Framework.
3. Install Influxdb.
4. Install Docker.
   a)Conda setup
 
## Run

1. Execute `npm install`.
2. Execute `npm start`.
